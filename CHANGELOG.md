# Changelog

## 0.1.0 (2026-07-16)

### ✨ Features
- Pipeline complète : extraction → analyse → export
- Extracteur PDF (PyMuPDF + pdfplumber)
- Extracteur DOCX (python-docx)
- Extracteur PPTX (python-pptx)
- Extracteur HTML (lxml)
- Extracteur Markdown (markdown-it-py)
- Analyseurs : chapitres, titres, langue, code, formules, références, statistiques
- Exporters : JSON, Markdown, HTML
- CLI Typer (analyze, extract, list)
- Architecture plugin (TipTap externe)
- Modèles : Document, Chapter, Image, Table, CodeBlock, Formula, Link
